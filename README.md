# ALWAYSCHECKOUT INC. Corporate Website

Minimal single-page corporate website for ALWAYSCHECKOUT INC.

This is a static HTML website designed for free hosting on Vercel, GitHub Pages, Cloudflare Pages, Netlify, or any ordinary static hosting service.

## Deployment

### Vercel

Import this repository in Vercel. No build command is required.

- Framework Preset: Other
- Build Command: leave empty
- Output Directory: leave empty

### GitHub Pages

Push to `main` or `master`. The included GitHub Actions workflow deploys the repository root as a static site.

## Files

- `index.html` - production website
- `vercel.json` - Vercel static hosting configuration
- `.github/workflows/pages.yml` - GitHub Pages deployment workflow
- `squarespace-snippet.html` - optional Squarespace code-injection version
